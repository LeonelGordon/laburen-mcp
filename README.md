# Laburen MCP

MCP desarrollado para el desafío técnico de AI Engineer en Laburen.

Permite a un agente de IA:
- Explorar productos
- Crear y gestionar carritos de compra
- Integrarse con Chatwoot para derivación a agente humano

## Arquitectura
- Cloudflare Workers + MCP
- Base de datos D1
- Agente IA en plataforma Laburen

## Documentación
La documentación completa y el diagrama de flujo se encuentran en la carpeta `/docs`.
